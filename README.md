✅ **#Project Overview**
The Meghplat Homepage is a dynamic and responsive website built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). It serves as the landing page for the Meghplat platform, offering a clean and user-friendly interface with smooth navigation and modern design.


**🌐 Key Feature**s
Responsive Design:

Fully optimized for mobile, tablet, and desktop devices.
Clean and modern UI with consistent color themes.
Navigation and Routing:

Seamless navigation with React Router.
Easy access to different sections of the platform.
API Integration:

Connected with backend APIs for dynamic content display.
Optimized Performance:

Fast loading time due to Netlify deployment.

**🔥 Installation and Setup**
To run the project locally:

**Clone the repository:**

git clone https://github.com/Hariomkabra/Meghplat.git

**Navigate to the project directory:**

cd Meghplat

**Install dependencies:**

npm install

**Start the server:**

npm start

**The app will be running at:**

👉 http://localhost:3000
